# 用 Remotion 打造 SwiftClip — 開發者心得分享

用 React 寫影片，這件事聽起來很瘋，但 **Remotion** 讓它變得超級合理。

我用 Remotion 開發了 **SwiftClip**，一個可以線上瀏覽、預覽、並即時渲染動態影片模板的平台。總共做了 30+ 個模板，包含標題動畫、數據視覺化、社群 Story、產品發布等各種場景。

🔗 線上體驗：https://swift-clip.vercel.app/
💻 開源程式碼：https://github.com/zz41354899/SwiftClip

---

**為什麼我推薦 Remotion？**

- **開發體驗極佳** — 熱重載預覽、TypeScript 支援、組件化 API，迭代速度非常快
- **伺服器端渲染** — `@remotion/renderer` 讓我可以用一個 API endpoint 按需渲染影片，是 SwiftClip 匯出功能的核心
- **組合式設計** — 把影片拆成小元件重複使用，程式碼整潔又好維護

如果你是前端開發者，又想做影片相關的產品，Remotion 絕對值得試試。
